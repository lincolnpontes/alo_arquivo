# Alô Arquivo

Aplicativo web instalável para leitura contínua de QR Code e códigos de barras.

## O que ele faz

- mantém a câmera aberta durante várias leituras;
- salva cada código apenas uma vez;
- numera a lista automaticamente;
- mantém a lista no próprio aparelho;
- compartilha os códigos, um por linha, pela tela nativa do Android/iOS;
- funciona como PWA e pode ser instalado na tela inicial.

## Publicação no GitHub Pages

Publique os arquivos desta pasta na raiz do repositório e ative o **GitHub Pages** nas configurações. O acesso por HTTPS é necessário para o navegador liberar a câmera.

Na primeira abertura, o app precisa de internet para carregar o mecanismo de leitura. Depois que ele for armazenado pelo navegador, o restante do app permanece disponível offline.
